ℹ️ 注意事项

坐标系：转换后的 GeoJSON 默认为 WGS84 坐标系（EPSG:4326）

环方向：ArcGIS 外环顺时针，GeoJSON 外环逆时针，库会自动处理

ID 映射：默认使用 OBJECTID 或 FID 作为要素 ID

三维坐标：支持 Z 坐标（高程）的转换
